# Overview
This project implements a machine learning model designed to predict healthcare outcomes with input from domain experts. It includes data preprocessing, model training, fine-tuning, and evaluation notebooks.


## Directory Structure
- `data/` - Contains patient datasets.
- `notebooks/` - Contains Jupyter notebooks for evaluation and fine-tuning.
- `src/` - Python scripts for preprocessing, modeling, and utility functions.


## Requirements
- Python 3.9+
- scikit-learn
- pandas
- numpy
- matplotlib
- seaborn
- xgboost
- shap
